# Development Environment Instructions

This development container is pre-configured with the following tools and technologies to streamline your workflow:

---

## 🐍 Python Development

- **Python 3**:

  - An up-to-date version of `python3` is pre-installed and available on the `PATH`.
  - Use `python3` for running Python scripts.

- **pip3**:

  - The Python package manager `pip3` is pre-installed and available on the `PATH`.
  - Use `pip3` to install Python dependencies.

- **Python Extensions**:
  - The development container includes Python language extensions for enhanced development in VS Code.

---

## 🐙 Git

- **Git**:
  - An up-to-date version of Git is pre-installed and available on the `PATH`.
  - Built from source if necessary to ensure the latest version.

---

## 🟢 Node.js Development

- **Node.js**:

  - A recent version of `node` is pre-installed and available on the `PATH`.

- **npm**:

  - The Node.js package manager `npm` is pre-installed and available on the `PATH`.

- **ESLint**:
  - `eslint` is pre-installed for linting JavaScript and Node.js code.

---

## 🛠️ Usage

- **Python**:

  - Run Python scripts:
    ```bash
    python3 script.py
    ```
  - Install dependencies:
    ```bash
    pip3 install -r requirements.txt
    ```

- **Git**:

  - Use Git for version control:
    ```bash
    git clone <repository-url>
    git commit -m "Your commit message"
    ```

- **Node.js**:

  - Run JavaScript files:
    ```bash
    node script.js
    ```
  - Install Node.js dependencies:
    ```bash
    npm install
    ```

- **ESLint**:
  - Lint JavaScript files:
    ```bash
    eslint file.js
    ```

---

This environment is ready for Python, Node.js, and Git-based development. If you encounter any issues, ensure the tools are correctly configured in your project.
